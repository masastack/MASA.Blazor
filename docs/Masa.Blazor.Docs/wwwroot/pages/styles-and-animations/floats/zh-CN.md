# Float（浮动）

使用响应式 float 工具在任何断点上应用自定义浮动。

### 其他

#### 类

**概述**

浮动辅助类基于当前视口尺寸(及以上)使用 [CSS 浮动属性](https://developer.mozilla.org/en-US/docs/Web/CSS/float) 应用浮动功能.

Floats 也可以在每个断点（视图）的基础上适用。

* **.float-right**
* **.float-start**

* **.float-sm-right**

* **.float-sm-end**
* **.float-sm-none**
* **.float-md-left**
* **.float-md-right**

* **.float-lg-left**

* **.float-xl-right**

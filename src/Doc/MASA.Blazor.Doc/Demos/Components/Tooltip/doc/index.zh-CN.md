当用户悬停在元素上时， `MTooltip` 组件可用于传递信息。 您还可以通过 **@bind-Value** 来控制提示的显示。 当激活时，提示将显示用于标识元素的文本，例如其功能的描述。

## API

- [MTooltip](/api/MTooltip)

## 注意

<!--alert:info-->
为了正确定位 `MTooltip`，需要一个位置支撑(`Top` | `Bottom` | `Left` | `Right`)。

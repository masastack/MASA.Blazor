`MIcon` 组件提供了一系列用于为您的应用程序的各个方面提供上下文的胶卷。 对于所有可用图标的列表, 请访问官方的 [Material Design 图标](https://materialdesignicons.com/) 页面。
要使用任何这些图标，只需使用 `mdi-` 前缀，然后是图标名称。

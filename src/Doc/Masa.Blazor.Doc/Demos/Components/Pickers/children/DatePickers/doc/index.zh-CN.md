`MDatePicker` 是一个功能齐全的日期选择组件, 它让用户选择一个日期或日期范围。

## API

- [MDatePicker](/api/MDatePicker)

## 注意

<!--alert:warning--> 
`MDatePicker` 接受ISO 8601 **日期** 字符串(YYYY-MM-DD)。 有关 ISO 8601 和其他标准的更多信息，请访问 ISO（国际标准化组织） [国际标准](https://www.iso.org/standards.html) 官方网页。

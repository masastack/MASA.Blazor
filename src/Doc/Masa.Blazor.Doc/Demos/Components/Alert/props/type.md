## zh-CN

**Type** 属性提供 4 种默认的样式：**Success**, **Info**, **Warning**, 和 **Error**。每个样式都提供默认图标和颜色。

## en-US

The **Type** prop provides 4 default styles: **Success**, **Info**, **Warning**, and **Error**. Each of these styles
provide a default icon and color.

## zh-CN

分页默认根据设置的 **Length** 属性显示页数，两边有 **Prev** 和 **Next** 按钮帮助导航。

## en-US

By default, paging displays the number of pages according to the set **Length** property, and there are **Prev** and **Next** buttons on both sides to help navigation.

## zh-CN

边框半径可以通过使用 **t**, **r**, **b**, **l** 内置类在每条边上配置; 例如. `.rounded-b-xl` 和 `.rounded-t`。

## en-US

The border radius can be configured on each side by using **t**, **r**, **b**, **l** built-in classes; e.g. `.rounded-b-xl` and `. rounded-t`.

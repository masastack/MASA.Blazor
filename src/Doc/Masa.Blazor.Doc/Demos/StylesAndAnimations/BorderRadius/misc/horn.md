## zh-CN

边框半径可以通过使用 **tl**, **tr**, **br**, **bl** 内置类在每个角上配置; 例如. `.rounded-br-xl` 和 `.rounded-tr`。

## en-US

The border radius can be configured on each corner by using **tl**, **tr**, **br**, **bl** built-in classes; e.g. `rounded-br-xl` and `. rounded-tr`.
